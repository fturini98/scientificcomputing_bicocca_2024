class BicoccaCoursePython2024.prima_lezione.tic_tac_toe[source]

Bases: object

A class to represent the Tic-Tac-Toe game.

The class allows two players to play Tic-Tac-Toe, alternating between ‘X’ and ‘O’. It handles the initialization of the board, making moves, checking for a winner, and determining whether the game continues or ends.

check_game()[source]

Check the current game state for a winner or tie.

This method checks the rows, columns, and diagonals for a winning combination. If all squares are filled and no winner, it results in a tie.

get_move(n: int, xo: str)[source]

Ask the current player to make a move. Ensures the selected square is empty.

Parameters:

  • n (int) – The current player’s number (1 or 2).

  • xo (str) – The player’s mark (‘X’ or ‘O’).

This method will continue to ask for input until a valid move is made.

initialize_board()[source]

Initialize the game board with empty spaces for each square (s1 to s9).

play_game()[source]

Play a game of Tic-Tac-Toe, alternating turns between two players.

This method runs the game loop, asking players to make moves, updating the board, and checking for a winner after each turn.

show_board()[source]

Display the current game board using the dictionary self.play to populate the board template.
